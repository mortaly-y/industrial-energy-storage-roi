# 工商业储能柜现场调试与验收 SOP

## 1. SOP 目的

本 SOP 用于模拟工商业储能项目现场调试与验收流程，覆盖到货前资料确认、上电前检查、通讯联调、功能测试、策略验证和验收资料归档。

## 2. 到货前资料确认

| 检查项 | 内容 | 结果 |
|---|---|---|
| 技术协议 | 功率、容量、接入方式、运行模式是否明确 | 待确认 |
| 一次图 | 储能接入点、开关柜、变压器、母线关系是否明确 | 待确认 |
| 二次图 | 控制回路、急停、消防、联锁逻辑是否明确 | 待确认 |
| 通讯拓扑 | EMS、PCS、BMS、电表、消防、温控通讯链路是否明确 | 待确认 |
| 通讯点表 | 点位、地址、数据类型、字节序、倍率是否确认 | 待确认 |
| 设备清单 | 电池柜、PCS、EMS、电表、消防、温控、交换机是否齐套 | 待确认 |
| EMS 策略参数 | SOC 上下限、峰谷时段、防逆流、需量阈值是否确认 | 待确认 |

## 3. 上电前检查

1. 检查设备外观是否损坏；
2. 检查柜体接地是否可靠；
3. 检查交流侧接线是否正确；
4. 检查直流侧极性是否正确；
5. 检查绝缘测试结果是否满足要求；
6. 检查 CT/PT 接线方向和变比；
7. 检查通讯线屏蔽层接地；
8. 检查消防、温控、门禁、急停状态；
9. 检查 PCS、BMS、EMS 参数版本；
10. 确认现场具备安全送电条件。

## 4. 通讯联调

| 通讯链路 | 检查内容 | 判断标准 |
|---|---|---|
| EMS ↔ PCS | 功率、状态、故障、启停、功率设定 | 数据正常，控制可下发 |
| EMS ↔ BMS | SOC、SOH、电压、温度、告警 | 数据与本地屏一致 |
| EMS ↔ 电表 | 并网点功率、电压、电流、电能 | 功率方向正确 |
| EMS ↔ 消防 | 消防状态、动作信号 | 告警可上传 |
| EMS ↔ 温控 | 温度、运行状态、故障 | 数据正常 |
| EMS ↔ 云平台 | 运行数据、告警、曲线 | 云端可查看 |

## 5. 功能测试

### 5.1 手动充电测试

1. 设置 PCS 为手动模式；
2. 下发小功率充电指令；
3. 检查 PCS 功率方向；
4. 检查 SOC 是否上升；
5. 检查电表功率是否增加；
6. 检查 EMS 曲线是否记录。

### 5.2 手动放电测试

1. 下发小功率放电指令；
2. 检查 PCS 功率方向；
3. 检查 SOC 是否下降；
4. 检查并网点功率是否下降；
5. 检查是否存在反送电风险。

### 5.3 自动策略测试

1. 设置峰谷时段；
2. 设置 SOC 上下限；
3. 启用谷充峰放策略；
4. 验证谷段自动充电；
5. 验证峰段自动放电；
6. 验证 SOC 越限停止；
7. 验证故障状态停机。

## 6. 常见问题与处理

| 问题 | 可能原因 | 处理方式 |
|---|---|---|
| EMS 读不到 PCS | IP 地址、端口、设备地址、通讯协议错误 | 检查通讯参数和网络连通性 |
| SOC 显示异常 | 寄存器偏移、字节序、倍率错误 | 对照 BMS 协议重新解析 |
| 电表功率方向反 | CT 方向或功率符号定义错误 | 调整 CT 方向或修正符号 |
| PCS 不响应指令 | 控制模式错误、权限未开放、故障锁定 | 切换控制模式并复位故障 |
| 策略不执行 | 时间段、SOC 限值、设备状态不满足 | 检查 EMS 策略参数 |
| 云端无数据 | 网络异常、平台地址错误、网关未上线 | 检查网络与平台配置 |

## 7. 验收资料清单

1. 技术协议；
2. 设备清单；
3. 一次图；
4. 二次图；
5. 通讯拓扑图；
6. 通讯点表确认表；
7. EMS 参数设置表；
8. 调试记录；
9. 充放电测试记录；
10. 告警测试记录；
11. 运行曲线截图；
12. 客户培训记录；
13. 问题闭环表。

## 8. 项目价值

解决方案工程师不能只写方案，还要考虑方案到了现场能否调通、能否验收。本 SOP 将方案设计、通讯点表、EMS 策略和现场交付连接起来，用于降低现场返工和沟通成本。
