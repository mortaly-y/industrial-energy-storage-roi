# 项目背景与客户需求

## 1. 项目定位

本项目为面向储能解决方案工程师岗位的作品集项目，模拟某制造企业用户侧工商业储能方案设计过程。

项目重点不是展示单一技术点，而是覆盖储能解决方案工程师常见工作链路：

客户需求分析 → 负荷与电价分析 → 储能容量配置 → 收益测算 → EMS 策略设计 → 通讯点表梳理 → 调试验收 SOP。

## 2. 客户背景

客户为某制造企业，生产负荷存在明显日内波动。白天生产高峰期用电负荷较高，峰段电价压力明显；夜间负荷较低，谷段电价较低，具备用户侧储能削峰填谷和峰谷套利的基本条件。

## 3. 客户核心痛点

1. 峰段电价较高，综合用电成本持续上升；
2. 最大负荷接近合同容量上限，存在需量电费增加风险；
3. 生产负荷存在波动，希望通过储能平滑并网点功率；
4. 客户缺乏储能项目经验，需要明确系统容量、投资收益和运行策略；
5. 后续项目需要形成可实施、可调试、可验收的技术方案。

## 4. 初步方案目标

本项目初步配置 500kW/1MWh 用户侧储能系统，接入厂区低压母线侧，通过 EMS 统一调度 PCS、BMS、电表等设备，实现以下目标：

- 谷段充电；
- 峰段放电；
- 削峰填谷；
- 需量控制；
- 防逆流；
- SOC 上下限保护；
- 异常状态停机保护。

## 5. 方案输出物

本项目输出以下内容：

1. Python 收益测算模型；
2. 24 小时负荷与电价样例数据；
3. 收益测算 Excel；
4. 负荷曲线、SOC 曲线、充放电功率曲线；
5. EMS 控制策略说明；
6. EMS-BMS-PCS 通讯点表；
7. 储能柜现场调试验收 SOP；
8. 面试项目讲解稿。
