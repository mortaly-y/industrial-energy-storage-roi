# EMS 控制策略说明

## 1. EMS 在储能系统中的角色

EMS，即 Energy Management System，能量管理系统，是工商业储能系统的调度核心。

在本项目中，EMS 主要负责采集电表、PCS、BMS 等设备数据，并根据电价、负荷、SOC、设备状态和安全约束，向 PCS 下发充放电控制策略。

## 2. 核心输入数据

| 数据来源 | 关键数据 | 用途 |
|---|---|---|
| 电表 | 并网点有功功率、电压、电流 | 判断负荷状态、防逆流、需量控制 |
| PCS | 有功功率、运行状态、故障状态 | 判断变流器运行情况 |
| BMS | SOC、SOH、单体温度、单体电压、告警状态 | 判断电池安全边界 |
| 电价表 | 峰、平、谷时段及价格 | 决定充放电时间 |
| EMS 参数 | SOC 上下限、功率限制、运行模式 | 形成调度约束 |

## 3. 削峰填谷策略

### 谷段充电

当满足以下条件时，EMS 允许储能充电：

1. 当前电价类型为谷段；
2. SOC 低于上限；
3. BMS 无严重告警；
4. PCS 处于可运行状态；
5. 并网点容量允许充电。
